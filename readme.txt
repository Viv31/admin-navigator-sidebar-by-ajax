== Short Description ==
A plugin that enhances the WordPress admin dashboard navigation by enabling AJAX-based operations to prevent full page reloads and reduce server load.

== Description ==
The AJAX Admin Navigation Sidebar plugin improves the WordPress admin experience by enabling AJAX-based navigation. It reduces server load by preventing unnecessary page reloads. Once activated, it seamlessly converts all main admin navigation into AJAX-based transitions.

### Features:
1. Admin login with AJAX.
2. Admin logout with AJAX.
3. Edit admin profile using AJAX.
4. Smooth admin navigation using AJAX.
5. Internal admin dashboard navigation with AJAX.
6. Filters with AJAX functionality.

This plugin also disables right-click functionality and certain keyboard shortcuts for better security.

== How to Use ==
After installation and activation, the plugin automatically enables AJAX for the main admin navigation. No additional configuration is required.

== Requirements ==
- WordPress version 4.6 or higher.
- PHP version 5.2.4 or higher.

== Frequently Asked Questions ==

= How does this plugin work? =
This plugin replaces traditional navigation in the WordPress admin dashboard with AJAX-based navigation. It uses an AJAX loader to display content dynamically without reloading the page.

= Why should I use this plugin? =
This plugin minimizes server load and improves user experience by eliminating full-page reloads during admin navigation.
