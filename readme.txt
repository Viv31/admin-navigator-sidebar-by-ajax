== Description ==
Adds AJAX navigation to the WordPress admin dashboard to prevent full page reloads.

== Short Description ==

== How to Use ==
After installation, the plugin will automatically convert main navigation to the AJAX naigation.

== Requirements ==
It can run on any WordPress version.

== Frequently Asked Questions ==

= How this plugin works? =
IT makes all the main navigation with ajax loader in the WP admin area.

= Why use this plugin ? =
This plugin use ajax for displaying all the navigation menus that preents extra server load and page refresh.
